# Logo Espoir Vie ASBL

## Fichiers requis

Pour que le logo fonctionne correctement sur le site, veuillez placer les fichiers suivants dans ce dossier (`public/assets/img/`) :

1. **`logo-espoir-vie.svg`** - Logo principal (bleu) pour le header et la sidebar
2. **`logo-espoir-vie-white.svg`** - Logo blanc pour le footer

## Format du logo

Le logo doit être au format **SVG** avec :
- Couleur principale : Bleu (#2563EB)
- Texte : "Espoir vie" en style script/handwritten
- Format : Vectoriel (SVG) pour une qualité optimale à toutes les tailles

## Structure du logo

D'après la description fournie :
- Trois figures humaines stylisées autour d'un globe
- Deux arcs courbes encerclant les figures
- Texte "Espoir vie" en écriture manuscrite en dessous
- Fond transparent

## Remplacement des anciens logos

Les anciens fichiers `logo.svg` et `logo-white.svg` peuvent être conservés ou supprimés, car ils ne sont plus utilisés.
